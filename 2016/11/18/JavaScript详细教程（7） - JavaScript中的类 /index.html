JavaScript详细教程（7） - JavaScript中的类

发表于 2016-11-18   |   分类于 JavaScript

在JavaScript中，类的实现是基于原型集成机制的。如果两个对象实例都是从同一个原型对象上继承属性，我们说他们是同一个类的实例。JavaScript类的一个重要的特性就是“动态可继承”

使用构造函数的方式创建对象

使用工厂函数创建对象的方法并不常用，使用构造函数的方式创建对象更加常见，使用构造函数实现继承的最重要的特征就是构造函数的prototype属性被用做新对象的原型，这意味着通过同一个构造函数创建的所有对象都集成自一个相同的对象，因此它们都是同一个类的成员，如西面代码所以，使用构造函数代替工厂函数

构造函数函数的命名规则（首字母大写）和普通函数有所区分的原因是构造函数调用和普通函数调用不尽相同，构造函数是用来创建新对象的，它必须通过关键字new调用，如果将构造函数用做普通函数往往不能正常工作

prototype属性

原型对象是类的唯一标识，当且仅当两个对象继承自同一个原型对象时，它们才是属于同一个类的实例。而初始化对象的状态的构造函数则不能作为类的标识，因为两个构造函数的prototype属性可能指向同一个原型对象。那么两个构造函数创建的实例是属于同一个类。当使用instanceof运算符来检测对象是否属于某个类时会用到构造函数，假设这里有一个对象r，若想判断r是否是Range对象，可以使用如下代码：
